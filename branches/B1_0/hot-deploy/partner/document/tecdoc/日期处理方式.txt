1.basiccod.js添加全局设置
/** 
 * 修改date对象数据的JSON提交方式 
 */  
Ext.JSON.encodeDate = function(d) {  
	time=d.getTime();
    return   time;
};  
每次提交数据的时候会把日期转换成long格式


2.编辑界面model日期字段定义如下：
{name: 'bizDate',  type: 'date' ,
	convert: function(value, record) {
		return new Date(value);
	}
}

convert定义转换方式，从服务器请求的时间数据格式是long类型，需要通过该函数进行转换


3.列表界面model日期字段定义
{name: 'bizDate',type:'date',dateFormat:'time'}

4.列表界面gridpanel dateColumn定义
{
      xtype: 'datecolumn',
      dataIndex: 'bizDate',
      width:150,
      format : 'Y-m-d',
      groupable: false,
      text: '业务日期'
}
